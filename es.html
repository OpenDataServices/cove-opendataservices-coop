Estándar de Datos de Contrataciones Abiertas

El Estándar de Datos de Contrataciones Abiertas promueve el uso eficaz de los datos de contrataciones, ayudando a los usuarios a "seguir el dinero", y proporciona una plantilla clara para los gobiernos que deseen dar a conocer sus datos.


Herramienta de Datos de Contrataciones Abiertas

La creación y uso de Datos Abiertos se hace más fácil cuando hay buenas herramientas para ayudar.

CoVe existe para ayudar a la gente a:

  • Convertir datos entre formatos comunes (por ejemplo, csv a json)
  • Validar datos contrastándolos con las reglas
  • Explorar los datos que las máquinas pueden leer fácilmente, pero humanos no tanto

¿Qué sucede con los datos que proporciono a este sitio?

Conservamos los datos que cargas o pegas a este sitio en nuestro servidor durante 7 días.

Si proporciona un enlace, buscamos los datos de ese enlace y los almacenamos en nuestro servidor durante 7 días.

Borramos todos los datos de más de 7 días desde nuestros servidores diariamente, no conservamos ninguno de los datos originales.

Mientras los datos están en nuestros servidores podemos acceder a esos datos para ayudarnos a entender cómo las personas usan esta aplicación, lQué tipo de datos se están suministrando, los errores comunes que existen, etc.

También podemos conservar los datos en las copias de seguridad de nuestros servidores, lo que significa que algunos datos pueden ser retenidos por más tiempo. No tenemos ninguna intención de utilizar estos datos para nada que no sea la recuperación del servidor en caso de emergencia.

Guardamos algunos metadatos sobre los datos suministrados a este sitio. Los detalles se pueden encontrar en el código, pero pueden incluir información sobre si el archivo fue subido, vinculado o pegado, el tamaño del archivo, la fecha / hora en que se suministra y así sucesivamente.

¿Por qué eliminan los datos después de 7 días?

Este es un servicio que permite a la gente a explorar datos legibles por computadora. Como tal no vemos la necesidad de almacenar y reunir todo lo que la gente subre al sitio por siempre. Hemos optado por 7 días como un tiempo práctico que permite a la gente compartir los resultados con colegas, pero también permite que los datos desaparezcan con el tiempo para ahorrar a las personas tener que hacer una limpieza ellos mismos.

Creemos que la supresión de los datos suministrados después de 7 días proporciona un nivel de privacidad para los usuarios de este servicio.

¿Por qué ofrecer versiones convertidas?

El W3C Datos sobre Mejores Prácticas en la Web recomienda poner los datos abiertos disponibles en una variedad de formatos para satisfacer las necesidades de los distintos usuarios, por ejemplo, los desarrolladores pueden requerir JSON, los investigadores podrían preferir un formato de hoja de cálculo.
